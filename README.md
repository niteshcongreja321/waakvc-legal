## Custom domain DNS

Point `niteshkumar.dev` to GitHub Pages:

**A records** (apex domain):

```
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
```

**Optional** `www` CNAME:

```
www → niteshcongreja321.github.io
```

After DNS propagates, enable HTTPS in the repository's GitHub Pages settings if it is not already enforced.
